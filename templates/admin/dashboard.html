Estado y documentación de la API

🔌 API REST — MiniLog

Permite registrar QSOs desde software externo (WSJT-X, Log4OM, scripts, etc.)

GET /api/status

Verifica que la API esté activa. No requiere autenticación.

curl http://tu-servidor:5001/api/status
POST /api/qso

Registra un QSO. No requiere sesión — el operador se identifica con mycall.

curl -X POST http://tu-servidor:5001/api/qso \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer radioaficionado" \
  -d '{
    "call":      "K1ABC",
    "mycall":    "LU2FTI",
    "mode":      "FT8",
    "freq":      14.074,
    "extra":     "notas opcionales",
    "timestamp": "2026-04-25 14:00:00"
  }'

Header requerido: Authorization: Bearer radioaficionado
Campos requeridos: call, mycall (debe estar registrado en el sistema).
Opcionales: mode, freq (MHz), extra, timestamp (UTC, default: ahora).

GET /api/qso

Lista todos los QSOs del usuario autenticado.

curl -b cookies.txt http://tu-servidor:5001/api/qso
